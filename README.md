# CipherSQLStudio

CipherSQLStudio is a browser-based SQL learning platform where users can practice SQL queries against pre-configured assignments with real-time execution and intelligent hints.

## Features

- View SQL assignment questions
- Write and execute SQL queries in the browser
- Get AI-powered hints (without revealing the full solution)
- See query results instantly in a results table
- Clean and interactive SQL editor

## Tech Stack

Frontend:
- React (Vite)
- TailwindCSS
- Axios

Backend:
- Node.js
- Express.js

Database:
- PostgreSQL

AI Integration:
- OpenAI API for hint generation
